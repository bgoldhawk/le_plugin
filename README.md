# Podcast of the Lotus Eaters — Grayjay Plugin

A [Grayjay](https://grayjay.app) plugin for [lotuseaters.com](https://www.lotuseaters.com), a UK politics, culture and news platform.

## Features

- Browse latest content from the home feed
- Browse by category (Podcast, Documentary, Premium, etc.)
- Search across all content
- Play videos via Rumble embeds
- Premium content support (requires login)

## Installation

Import the plugin into Grayjay using the config URL:

```
https://raw.githubusercontent.com/bgoldhawk/le_plugin/main/LotusEatersConfig.json
```

## Premium Content

To access members-only content, tap the login button in the plugin settings and sign in with your Lotus Eaters account. After logging in you will be redirected to the home page — Grayjay will capture your session at that point.
